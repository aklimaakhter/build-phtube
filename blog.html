Question : Discuss the scope of var, let, and const.

Answer : var declarations are globally scoped or function scoped . let and const are declarations block scoped. var can be updated and re-declared ; let variables can be updated but not re-declared; const variables can not be updated or re-declared.

Question : Tell us the use cases of null and undefined.

Answer : Undefined means the variable declared, but not value assigned. Null means an empty value or a blank value.
